Welcome to Mercury HSX Reader

Compile your templates into Mercury compatible HSX virtual DOM DSL

  <:MyCool />
  <SideBySideEditor />
  <MyEditor state="sideBySideEditor"/>

  <div>
    <:SideBySideEditor />
  </div>

  <header>
    <section name="main">
      <h2>my title</h2>
      <p>some text here</p>
    <section>
  </header>

When compiled the markup is turned into mercury compatible (nested) h calls which creates a virtual dom.

this.renderComponent('MyCool', null)
this.renderComponent('SideBySideEditor', null)
this.renderComponent('MyEditor', { state: 'sideBySideEditor' })
h('div', null, [this.renderComponent('SideBySideEditor', null)])

h('header', null, 
  [h('section', { name: 'main' }, 
    [h('h2', null, 
       ['my title']), 
     h('p', null, 
       ['some text here']) 
    ])
   ]
)

Notice that currently, your render context is expected to contain a function renderComponent which is called with arguments that identify the component and arguments. It then has the responsibility to find the component and call it as per your conventions.

To mark a tag as a component call, either:

  • prefix its name with a :
  • capitalize the tag name

Examples: <MyEditor state="sideBySideEditor"/> and <:SideBySideEditor />
